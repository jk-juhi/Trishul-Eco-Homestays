# Project Brief

## Project Title

Trishul Eco Homestays – AI-Powered Guest Review Sentiment Analysis and Business Intelligence Dashboard

## Problem Statement

Trishul Eco Homestays receives guest feedback through various booking platforms and direct interactions. The volume of reviews makes it difficult to manually analyze customer sentiment, identify recurring issues, and derive actionable business insights.

Additionally, there is no centralized dashboard for monitoring guest satisfaction metrics and review trends.

## Project Objectives

1. Develop an AI-powered sentiment analysis system to classify guest reviews into Positive, Neutral, and Negative categories.
2. Identify key themes and trends within guest feedback.
3. Create an interactive Business Intelligence dashboard for visualizing review insights and performance metrics.
4. Provide actionable recommendations to improve guest experience and operational efficiency.

## Scope

### AI Module

- Review sentiment classification
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Sentiment visualization

### Business Intelligence Module

- KPI tracking
- Guest rating analysis
- Review trend monitoring
- Interactive dashboards and reports

## Expected Outcomes

- Automated sentiment classification system
- Centralized guest feedback analysis
- Interactive Power BI dashboard
- Data-driven recommendations for management
